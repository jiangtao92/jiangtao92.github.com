# jekyll环境搭建

> 环境 ubuntu 12.04.4

1. 安装 `Ruby` 环境:

	```
	sudo apt-get install ruby1.9.1-dev
	```

2. 安装 `jekyll`:

	```
	sudo gem install jekyll
	```

> 长时间没有反应的话,需要换源解决

```
gem sources --remove https://rubygems.org/
gem sources -a http://ruby.taobao.org/
gem sources -l
```
__确保其中只出现`http://ruby.taobao.org/`__
